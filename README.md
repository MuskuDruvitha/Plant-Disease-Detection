# Plant Disease Detection

A deep learning-based web application built with Streamlit that detects plant diseases from leaf images.

## Features
- Upload a leaf image
- Predict plant disease using a trained CNN model
- Display disease description
- Show symptoms
- Suggest prevention methods
- Provide management recommendations

## Tech Stack
- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- Pillow

## Run Locally

pip install -r requirements.txt
streamlit run app.py
